# 消防栓

参见：消防栓官方网站在新窗口打开

消防栓是主要的管理 bot，会初步审核加群申请，过滤超限玩家。消防栓也会生成群员列表，帮助管理员引导超限玩家至对应的进阶群。

消防栓也有一些信息查询功能，其中“ 今日高光、pptth ”等功能为消防栓独有。
